# ThinkBridge Foodie Delight Admin app
## Tech stack
  - React + typescript
  - Vite
  - JSON server for API

## Clone the App
  `git clone https://github.com/dk-alt/ThinkBridgeFoodieDelight.git`

## Run the application
```
  npm run install
  npm run dev
  npm run server
```
visit http://localhost:5173/


## Build the application
`npm run build`
copy the build folder to web server
